## 1. Платформа LPTracker — что это и что умеет

CRM-система со встроенным голосовым AI-ботом. Позволяет создавать автоворонки, запускать автообзвон, принимать входящие звонки через робота.

### Сильные стороны
- Всё в одном: CRM + телефония + голосовые роботы + лендинги
- Низкий порог входа — визуальный конструктор, не нужно программировать
- Быстрый старт: загрузил базу → настроил сценарий → запустил
- Встроенная аналитика звонков
- Российский сервис, серверы в РФ, поддержка на русском
- Дешевле чем связка amoCRM + отдельный сервис обзвона

### Слабые стороны
- Качество синтеза речи — голос часто звучит неестественно
- Ограниченное распознавание речи — хорошо понимает "да/нет", но сложные фразы обрабатывает плохо
- Техподдержка — частые жалобы на медленные ответы
- Периодические баги и сбои
- Устаревший интерфейс
- Нет готовой интеграции с GPT из коробки

### Что мы можем сделать лучше
- Использовать предзаписанный голос вместо синтеза — конверсия вырастает в 2-3 раза
- Делать сценарии короткими (3-5 шагов)
- Тестировать каждый сценарий 10+ раз перед запуском
- Слушать записи первых 20-30 звонков и корректировать скрипт

## 2. Сценарии разговоров для ниши дезинфекции

### Обязательная структура диалога
1. **Приветствие** — назвать компанию в первые 5 секунд
2. **Квалификация** — узнать проблему, тип помещения, площадь, срочность
3. **Презентация** — предложить решение, назвать преимущества
4. **Обработка возражений** — "дорого", "подумаю", "не помогло раньше"
5. **Закрытие** — записать на выезд специалиста
6. **Подтверждение** — зафиксировать дату, время, адрес

### Вопросы которые робот должен задать
| # | Вопрос | Зачем |
|---|--------|-------|
| 1 | Какая проблема: насекомые, грызуны, дезинфекция? | Определить услугу |
| 2 | Какой конкретно вредитель? | Подобрать метод |
| 3 | Тип помещения (квартира, ресторан, офис, склад)? | Влияет на цену |
| 4 | Площадь в м2 или количество комнат? | Расчёт стоимости |
| 5 | Как давно проблема? | Оценка масштаба |
| 6 | Пробовали ли обращаться раньше? | Понять ожидания |
| 7 | Есть ли дети, животные, аллергики? | Выбор препарата |
| 8 | Когда удобна обработка? | Планирование |
| 9 | Адрес объекта? | Логистика |

## 3. Обработка типичных возражений

### "Дорого / у других дешевле"
> «Наша стоимость включает гарантию до 12 месяцев. Если вредители вернутся — повторный выезд бесплатный. Многие кто выбирает дешёвые службы платят дважды. У нас результат с первого раза.»

### "Я подумаю / перезвоню"
> «Конечно. Но чем дольше откладывать — тем сильнее заражение и дороже обработка. Давайте отправлю расчёт в СМС — будет конкретная цифра для решения.»

### "Уже вызывали — не помогло"
> «Такое бывает при слабых препаратах или неправильном методе. Мы работаем профессиональными средствами. Плюс гарантия — если проблема вернётся, приедем повторно бесплатно.»

### "А вы сертифицированы? Препараты безопасны?"
> «Да, у нас все документы — лицензия и сертификаты на препараты. Всё разрешено для применения в жилых помещениях. После обработки мастер оставит акт и рекомендации.»

### "У нас дети / животные"
> «Используем препараты 4-го класса опасности — малоопасные. После обработки проветрить 2-3 часа. Мастер расскажет когда можно вернуться — обычно через 3-5 часов.»

### "Сам справлюсь"
> «Бытовые средства не уничтожают гнёзда и кладки яиц. Профессиональная обработка проникает в труднодоступные места и действует до нескольких месяцев.»

### "Нужно согласовать с руководством"
> «Конечно. Отправлю расчёт в СМС — так проще обсудить. Когда перезвонить для подтверждения?»

## 4. Лучшие промпты для AI-ботов

### Структура промпта
```
[РОЛЬ] — кто бот (имя, должность, компания)
[КОМПАНИЯ] — услуги, преимущества
[ЦЕЛЬ ЗВОНКА] — одна конкретная цель
[СТИЛЬ РЕЧИ] — как говорить
[СЦЕНАРИЙ] — пошаговый алгоритм
[ОГРАНИЧЕНИЯ] — что нельзя
[ВОЗРАЖЕНИЯ] — ответы на типичные отказы
[ЗАВЕРШЕНИЕ] — как заканчивать
```

### Пример промпта для Дез2.0 (исходящий B2B)

```
РОЛЬ:
Ты — Анна, менеджер компании "Дез2.0". Ты звонишь потенциальным
клиентам из сферы HoReCa и офисного бизнеса в Санкт-Петербурге.

О КОМПАНИИ:
- "Дез2.0" — дезинфекция, дезинсекция, дератизация для бизнеса
- СПб и Ленинградская область
- Работаем без остановки бизнеса, в нерабочее время
- Полный пакет документов для Роспотребнадзора
- Сертифицированные препараты 3-4 класса безопасности
- Контрольный визит через 7-14 дней бесплатно
- Работаем 24/7

ЦЕЛЬ ЗВОНКА:
Назначить бесплатный аудит помещения.

СТИЛЬ:
- Короткие фразы: 1-2 предложения за реплику
- Деловой но дружелюбный тон
- Обращение на "вы"
- Каждую реплику заканчивай вопросом
- Вместо открытых вопросов давай 2 варианта

ОГРАНИЧЕНИЯ:
- НЕ называй конкретные цены
- НЕ говори плохо о конкурентах
- НЕ признавайся что ты робот
- Максимум 2 попытки преодолеть возражение
- Если не понимаешь клиента — переведи на оператора
```

### Пример промпта для Дез2.0 (входящий)

```
РОЛЬ:
Ты — Анна, оператор компании "Дез2.0". Принимаешь входящие звонки.

ЦЕЛЬ:
Выяснить проблему, собрать данные, записать на выезд.

СЦЕНАРИЙ:
1. "Дез2.0, Анна, здравствуйте! Чем могу помочь?"
2. Узнать проблему: "Расскажите, что случилось?"
3. Тип объекта, площадь, адрес
4. Предложить выезд: "Специалист приедет, осмотрит и проведёт
   обработку. Стоимость рассчитает на месте."
5. Согласовать дату и время
```

## 5. Обязательные правила для голосовых ботов

### Длина ответа
- Максимум 2-3 предложения за реплику
- Первая реплика — не более 15 секунд
- Общая длительность звонка — до 2-3 минут

### Паузы
- После вопроса клиента пауза 0.5 секунды
- Слова-связки: "Да, понял", "Хорошо", "Так"
- Если клиент молчит 5+ секунд: "Аллё, вы меня слышите?"

### Запреты
- Не признаваться что робот
- Не обсуждать политику, религию
- Не давать медицинских советов
- Не спорить с клиентом
- Не использовать жаргон
- Не говорить "проблема" — говорить "ситуация"

### Для синтеза речи (TTS)
- Числа писать словами: "пять тысяч" вместо "5000"
- Адреса полностью: "улица Рубинштейна, дом двенадцать"
- Аббревиатуры раскрывать
- Запятые для пауз, "..." для длинных пауз

## 6. Обязательные элементы для ниши дезинфекции

### Юридические
- Название компании — в первые секунды
- Лицензия и сертификаты — при сомнениях клиента
- Гарантия с конкретным сроком
- Акт выполненных работ (критично для юрлиц и общепита)

### Безопасность
- Класс опасности препаратов
- Время проветривания
- Вопрос про детей, животных, аллергиков
- Рекомендации после обработки

### Срочность
- "Чем раньше — тем проще и дешевле. Насекомые размножаются быстро"
- "Свободные окна на сегодня/завтра ещё есть"

### Сегментация клиентов
- **Физлица** — акцент на безопасности, гарантии, скорости
- **Юрлица (B2B)** — акцент на документах, регулярном обслуживании, СанПиН
- **Управляющие компании** — абонентское обслуживание

## 9. Рекомендации по улучшению (взгляд пользователя)

### 1. Спрашивать "удобно ли говорить?"
Робот сразу начинает разговор — а клиент может быть занят. После приветствия добавить: "Удобно сейчас буквально на минутку?" Если нет — предложить перезвонить. Иначе клиент просто бросает трубку.

### 2. Давать "крючок" в начале разговора
"Вы оставляли заявку по поводу непрошеных гостей" — непонятно если клиент забыл или это холодный звонок. Для холодных: начинать с боли — "Звоним потому что у ресторанов в вашем районе сейчас сезон тараканов." Для заявок: уточнить — "Вы оставляли заявку на сайте dez20.ru, всё верно?"

### 3. Называть хотя бы вилку цен
Клиент спрашивает "сколько?" — робот уходит от ответа. Это раздражает. Добавить: "Для кафе до ста квадратов — обычно от трёх до семи тысяч, точную цену скажет специалист на месте." Снимает тревогу и повышает доверие.

### 4. Отправлять SMS после записи
Клиент через час не помнит когда приедут и куда звонить. После записи — автоматическое SMS: "Дез2.0 — ваш специалист приедет 04.04 в 10:00. Адрес: ул. Рубинштейна 12. Вопросы: +7(812)627-15-50." Это есть в LPTracker — нужно настроить.

### 5. Научить робота говорить "я не знаю"
На нестандартные вопросы робот выдумывает общие фразы — это подрывает доверие. Добавить правило: если вопрос за рамками скрипта — честно сказать: "Это лучше обсудить со специалистом, он расскажет подробно при встрече. Записать вас?"

| Дифференциация | Язык «экспертов»: технологии, индивидуальный подход. Никаких «акций 50%» |

### 3.2 CSS-настройки

```css
/* Заголовки — вес 400, НЕ bold! Лёгкость = люкс */
h1 {
  font-family: 'Playfair Display', serif;
  font-weight: 400;
  font-size: clamp(2.5rem, 5vw, 4.5rem);
  letter-spacing: -0.02em;
  line-height: 1.1;
}

/* Лейблы — разрядка, uppercase. Hermes, Cartier, Rolex — все так делают */
.label {
  font-size: 0.75rem;
  letter-spacing: 0.15em;
  text-transform: uppercase;
  font-weight: 600;
}

/* Основной текст */
body {
  font-family: 'Inter', sans-serif;
  font-weight: 400;
  font-size: 1rem;
  line-height: 1.7;     /* Просторный = дорого */
}
```

**Правило:** font-weight 300–400 для заголовков. Жирный шрифт (700–900) выглядит «дёшево». Исключение — заголовок 80px+, тогда допустим 500.

## ЧАСТЬ 4: СИСТЕМА ОТСТУПОВ

Пустое пространство = роскошь. Как бутик Louis Vuitton vs переполненный рынок.

```css
/* Секции — ОГРОМНЫЕ отступы */
section {
  padding: clamp(80px, 12vw, 160px) 0;
}

/* Контейнер */
.container {
  max-width: 1200px;
  padding: 0 clamp(1.5rem, 4vw, 4rem);
}
```

**Правило:** Если обычный сайт — `padding: 60-80px`, премиальный — `padding: 140-200px`. Разница в 1.5-2 раза.

## ЧАСТЬ 14: ОШИБКИ, УБИВАЮЩИЕ ПРЕМИАЛЬНОСТЬ

1. Слишком много цветов — максимум 2 акцентных
2. Стоковые фото с Shutterstock — видно за километр
3. Roboto/Open Sans для заголовков — слишком обыденные
4. border-radius: 50px на кнопках — ассоциация с конструкторами
5. Яркие линейные градиенты — устарело
6. Параллакс на всём — один = стильно, пять = головокружение
7. Автозапуск видео со звуком — немедленный уход
8. Слайдер на hero — конверсия слайдеров ~0
9. Popup с формой через 3 секунды — раздражает
10. Фото насекомых крупно на главной — отталкивает

## ЧАСТЬ 15: РЕКОМЕНДАЦИИ ДЛЯ ПРОЕКТА «ПРЕМИУМ»

**Оптимальная структура страницы:**
1. Шапка: логотип + телефон крупно + «Заказать звонок»
2. Hero: эмоциональный заголовок + CTA + соцдоказательство
3. Полоса доверия: «X лет • Y обработок • Гарантия 12 мес • Лицензия»
4. Услуги: 4-6 карточек с иконками (НЕ фото насекомых)
5. Преимущества: безопасность, гарантия, технологии, скорость
6. Как мы работаем: 4-5 шагов
7. Отзывы: крупные цитаты с именами и компаниями
8. CTA-блок: «Оставьте заявку — перезвоним за 5 минут»
9. Футер: контакты, лицензии, мессенджеры

**Ключевой посыл:** Не «дезинсекция», а «профессиональная защита пространства». Продаём спокойствие и технологию, не «убийство жуков».

Шестиугольная сетка поверх hero — data:image inline SVG, не нужен файл. Технологичный паттерн без лишних HTTP-запросов.

**2. Dual-Color Accent (Cyan + Green)**
Два акцентных цвета: циан (#00e5ff) для UI-элементов и зелёный (#22c95e) для stats/data. Создаёт глубину и разграничивает типы информации.

**3. Radial Gradient Glows в Hero**
```css
.hero::after {
  background: radial-gradient(circle, rgba(0,229,255,0.06) 0%, rgba(34,201,94,0.03) 40%, transparent 70%);
}
```
Несколько "световых пятен" в углах hero — имитирует освещение без картинок.

**4. Monospace для Branding**
Логотип и телефон в JetBrains Mono — terminal/hacker эстетика. Уместно для "технологического" позиционирования.

**5. Two-Column Hero Grid**
```css
.hero-grid { display: grid; grid-template-columns: 1fr 1fr; gap: 60px; }
```
Hero разделён на текст (левый) и визуальный блок (правый). Современный layout, дающий место для "dashboard" или статистики справа.

**6. Stats в Hero**
```css
.hero-stat-num { font-family: var(--font-mono); font-size: 2rem; color: var(--green); }
```
Числа в моноширинном зелёном шрифте в самом герое — сразу показывает масштаб.

**7. Mobile Menu на весь экран**
```css
.mobile-menu {
  position: fixed; inset: 0;
  background: rgba(12,18,34,0.98); backdrop-filter: blur(20px);
  display: flex; align-items: center; justify-content: center; gap: 32px;
}
```
Мобильное меню по центру на весь экран — аналог премиум.html, но тёмный вариант.

**8. Neon Text-Shadow на hover**
```css
.nav-links a:hover { color: var(--cyan); text-shadow: 0 0 8px rgba(0,229,255,0.3); }
```
Ссылки светятся при наведении — минимальный эффект, максимальная атмосфера.

## Честный разбор от лица клиента

*Представьте: я обычный человек. Вечером обнаружил в шкафу клопов или увидел таракана на кухне. В панике открываю интернет. Вот что я вижу на этих сайтах и что чувствую.*

---

### 1. "Сразу ли понятно, чем занимается компания?" — первое впечатление

**Что работает:**
У большинства заголовок отвечает на вопрос сразу — "уничтожение тараканов, клопов, мышей в СПб". Это хорошо. dezinsektor.spb.ru прямо пишет цену в заголовке вкладки браузера ("от 1500₽") — сразу снимает страх "а вдруг заломят". ecodez.spb.ru говорит о "20 годах опыта" — слышна солидность.

**Что не работает:**
- Cleacom — позиционирует себя как клининговую компанию. Я ищу дезинсекцию, и первый экран меня смущает: "уборка квартир". Непонятно, это вообще те люди?
- Почти везде нет живой картинки происходящего — ни фото до/после, ни реальных специалистов за работой. Стоковые картинки тараканов крупным планом вызывают отвращение, а не доверие.
- Многие сайты используют аббревиатуры "СЭС", "ДДД-предприятие", "дератизация" — я, как обычный человек, не понимаю что это. Звучит бюрократически и холодно.

**Вывод для Дез2.0:** Первый экран должен за 3 секунды ответить: кто вы, где работаете, что конкретно делаете. Никаких аббревиатур без расшифровки.

---

### 2. "Могу ли я доверять этим людям?" — что вызывает доверие, что — сомнение

**Что вызывает доверие:**
- Конкретные цифры лет работы ("20 лет" у ecodez.spb.ru)
- Упоминание договора и официальных документов: "работаем по договору, выдаём акты"
- spb.dez-sluzhba.com пишет "Вернём деньги, если проблема останется!" — очень сильный сигнал. Звучит как уверенность в себе
- Гарантия на переобработку в 1-3 года

**Что вызывает сомнение:**
- Почти нигде нет фото реальной команды. Кто эти люди? Анонимные исполнители — это тревожно, ведь они придут ко мне домой
- Отзывы есть у всех, но выглядят шаблонно: "Всё отлично, рекомендую. Анна." Никакой конкретики — похоже на фейк
- Слово "официальная" в названии (типа "Официальная СЭС") — выглядит как попытка косить под государственную службу. Настораживает, не успокаивает
- Несколько сайтов выглядят как не обновлявшиеся с 2015 года — мелкий шрифт, таблицы из 2010-х, кнопки без нормального дизайна. Думаю: "Неужели эта компания ещё работает?"
- Агрессивные всплывающие окна через 3 секунды после входа ("Оставьте номер прямо сейчас!") — раздражают и отталкивают

**Вывод для Дез2.0:** Нужны фото реальных специалистов с именами, живые отзывы с деталями ("клопы пропали через 3 дня, дети вернулись вечером того же дня"), гарантия возврата денег сформулирована чётко и без мелкого шрифта.

---

### 3. "Сколько это стоит?" — как показаны цены, понятно ли

**Хорошие примеры:**
- spb.dez-sluzhba.com: прайс с градацией по типам помещений. Конкретные цифры — от комнаты до многокомнатных. Я могу сразу прикинуть.
- css-sp.ru: чёткий прайс — 1-комнатная от 2900 руб., 2-комнатная от 3400 руб. Понятно, не страшно.
- dezinsektor.spb.ru: цена "от 1500₽" в заголовке — минимум указан, уже не страшно звонить.

**Плохие примеры:**
- ecodez.spb.ru: "цена определяется после выезда специалиста". Как клиент — это пугает. Я представляю, как приедет мужик, всё осмотрит, назовёт космическую сумму, а я уже вынужден соглашаться.
- gup-ses.ru: три раздела (юрлица, дома, квартиры), но конкретных сумм нет. Зачем тогда вообще показывать разделы?
- Многие дают диапазоны "от 1500 до 15000 руб." — это почти бесполезно. Разброс слишком велик, страх неизвестности остаётся.

**Вывод для Дез2.0:** Нужен понятный прайс с разбивкой по конкретным задачам: тараканы в 1-комн. — столько, клопы в 2-комн. — столько. Калькулятор — хорошо, но только если он реально считает, а не просто собирает заявку.

---

### 4. "Что будет происходить?" — понятен ли процесс работы

**Хорошие примеры:**
- spb.dez-sluzhba.com: процесс описан в 6 шагах — звонок → осмотр → расчёт → обработка (1-4 ч.) → уборка → проверка с гарантией. Отлично, понимаю, чего ожидать.
- ecodez.spb.ru: 6 этапов, понятно и последовательно.
- css-sp.ru: 3 этапа — заявка → подготовка → обработка (до 1 часа, вернуться через 2 часа).

**Плохие примеры:**
- Большинство сайтов описывают *методы* (холодный туман, горячий туман, барьерная защита), а не *что произойдёт со мной как клиентом*. Мне неинтересно, что такое "холодный туман". Мне важно: когда уходить из квартиры? Когда можно вернуться? Что делать с едой? Нужно ли убирать вещи?
- Нигде не написано, нужно ли мне что-то готовить заранее. Это критический пробел — клиент в панике и не знает, что делать до приезда специалиста.

**Вывод для Дез2.0:** Процесс описывать с позиции клиента, а не исполнителя. "Вы позвонили → мы приехали через X часов → вы ушли на 2 часа → вернулись в чистую квартиру → через неделю мы позвоним проверить". И отдельный блок — "Как подготовиться: убрать еду, накрыть аквариум, уйти на 3 часа".

---

### 5. "Безопасно ли это для детей/животных?" — есть ли ответ на страх

**Это самый важный страх клиента — и он почти везде закрыт плохо.**

**Что есть:**
- Почти все пишут "безопасно для людей и животных" или "IV класс опасности"
- help-insects.ru: "Безопасно для детей и животных" — написано прямо, как отдельный тезис
- css-sp.ru: уточняют класс опасности препаратов — звучит как факт, а не просто слова

**Что не работает:**
- Фраза "безопасно" рядом с фразой "рекомендуем покинуть помещение на 3 часа" — противоречие. Клиент думает: "Если безопасно, зачем уходить?"
- Никто не объясняет, что будет с кошкой/собакой: выносить? Закрыть в комнате? Можно ли вернуться с ребёнком в тот же день?
- Нет ни слова о беременных женщинах — а это огромный страх у части аудитории
- "IV класс опасности — абсолютно безопасные" — это ложь языком. IV класс означает "малоопасные", а не "абсолютно безопасные". Умный клиент почувствует манипуляцию и закроет вкладку.

**Вывод для Дез2.0:** Нужен отдельный раздел "Безопасность" с конкретными ответами: что делать с кошкой, когда возвращаться с ребёнком, что с едой на кухне, можно ли при беременности. Без заклинаний "абсолютно безопасно" — лучше честно: "препараты малоопасны, но на время обработки лучше уйти — как при любой химической обработке".

---

### 6. "Как быстро приедут?" — сроки и срочность

**Хорошие примеры:**
- spb.dez-sluzhba.com: "Выезжаем в течение 30 минут" — конкретно и сильно
- help-insects.ru: "в день обращения" — хорошо
- ecodez.spb.ru: "решаем проблемы день в день"
- css-sp.ru: "срочный выезд в течение двух часов"

**Что не работает:**
- gup-ses.ru: "приедем в любое время" — звучит красиво, но не конкретно. Что это значит? Через час? Завтра?
- Почти никто не указывает часы работы на первом экране. Я звоню в воскресенье в 10 вечера — мне кто-нибудь ответит?
- Обещание "выезжаем за 30 минут" — клиенты понимают, что это нереально в СПб в час пик. Такие обещания уже не вызывают доверия, а вызывают скепсис. Лучше честно "в тот же день" или "в течение 2 часов".

**Вывод для Дез2.0:** Указать конкретные рамки: режим работы (например, 8:00–22:00 или круглосуточно), реальное время выезда. Если есть срочный выезд — выделить отдельно. Честность здесь ценнее громких обещаний.

---

### 7. "Что делать дальше?" — понятен ли следующий шаг

**Что работает:**
- У большинства есть кнопка "Заказать звонок" или "Оставить заявку"
- spb.dez-sluzhba.com: кнопки расставлены по всему сайту, встречаются в нужных местах

**Что не работает:**
- Слишком много вариантов действия одновременно: позвонить / заказать звонок / оставить заявку / написать в мессенджер / рассчитать стоимость. Я теряюсь и не делаю ничего.
- Формы длинные — просят имя, телефон, тип помещения, площадь, вид вредителя. Я в панике, у меня тараканы бегают по кухне — я не хочу заполнять анкету.
- Нет ни одного сайта, где написано "позвоните прямо сейчас — мы ответим за 1 минуту и запишем на ближайший слот". Везде просто "оставьте заявку" в никуда.

**Вывод для Дез2.0:** Один главный призыв к действию, максимально простой. Минимальная форма: только имя и телефон. Или вообще одна кнопка "Позвонить нам" с номером. И обязательно — что произойдёт после нажатия: "перезвоним за 2 минуты, договоримся о времени".

### 8. "Почему именно они?" — убедительны ли преимущества

**Типичные преимущества на всех сайтах (и они одинаковые у всех):**
- "Опытные специалисты"
- "Сертифицированные препараты"
- "Гарантия результата"
- "Работаем официально"
- "Бесплатный выезд/консультация"
- "Доступные цены"

**Проблема:** Эти слова стоят на каждом втором сайте. Они ничем не отличают одну компанию от другой. Я читаю их и думаю: "Ну конечно, а кто напишет 'неопытные специалисты и без гарантий'?"

**Что действительно выделяется:**
- Конкретные числа: "20 лет работы", "обработали 15 000 объектов", "гарантия 3 года"
- Уникальная гарантия spb.dez-sluzhba.com: "Вернём деньги, если проблема останется" — это смело и запоминается
- Анонимность (ecodez.spb.ru) — неожиданный, но рабочий аргумент: машина без опознавательных знаков, соседи не узнают

**Что не работает:**
- "Мы лучшие на рынке" — пустышка без доказательств
- Логотипы сертификатов без расшифровки: я не знаю, что такое "одобрено Роспотребнадзором", и зачем мне это в моей квартире

**Вывод для Дез2.0:** Преимущества должны быть конкретными и неожиданными. Не "опытные специалисты", а "специалист с опытом от 5 лет — можем показать удостоверение". Не "гарантия", а "если через месяц вы увидите хоть одного таракана — переделаем бесплатно или вернём деньги без разговоров".

### 9. Что раздражает и путает на таких сайтах

Честный топ-10 раздражителей от лица клиента:

1. **Всплывающие окна через 2-3 секунды** — я только зашёл, ещё ничего не прочитал, а мне уже лезет "оставьте номер и получите скидку". Закрываю сайт.

2. **Давление скидками** — "скидка только с 9 до 15" или "скидка 10% при заказе онлайн". Чувствую искусственное давление и манипуляцию.

3. **Непонятные термины** — "дезинсекция", "дератизация", "ДДД", "горячий туман", "барьерная защита", "IV класс опасности". Я не биолог и не химик. Объясните по-человечески что вы делаете.

4. **Цены с огромным разбросом** — "от 1500 до 15000 руб.". Бесполезно. Это не диапазон, это просто страх неизвестности в числах.

5. **Отзывы без деталей** — "Всё отлично, рекомендую. Мария, 2022." Это выглядит как фейк. Хочу прочитать: что было, что сделали, что стало.

6. **Сайт плохо открывается на телефоне** — я обнаружил клопов вечером, у меня в руках смартфон. Если текст мелкий и кнопки не нажимаются — закрываю.

7. **Фото тараканов крупным планом** — я и так в панике. Зачем мне ещё раз показывать то, от чего я хочу избавиться?

8. **Ощущение, что сайт не обновлялся годами** — старый дизайн, даты в отзывах "2017 год", битые картинки. Думаю: "Эта компания вообще ещё работает?"

9. **Нет чёткого ответа "когда приедете"** — пишут "в кратчайшие сроки", "оперативно". Что это значит? Час? День? Неделя?

10. **Форма заявки запрашивает слишком много** — площадь, адрес, тип вредителя, степень заражения. Я хочу просто позвонить и договориться, а не проходить анкетирование.

---

### 10. Что вызывает желание сразу позвонить или оставить заявку

Вот что реально работает как триггер для немедленного действия:

1. **Конкретная цена на первом экране** — вижу "2-комнатная квартира от 3400 руб." и у меня нет страха неизвестности. Рука сама тянется к телефону.

2. **Гарантия возврата денег** — почти нигде нет нормально сформулированного. Там где есть — доверие взлетает моментально. Это снимает главный страх "выброшу деньги и не поможет".

3. **"Приедем сегодня"** — я хочу решить проблему сейчас, не завтра. "Выезд в день обращения" — это триггер.

4. **"Звонок бесплатный, ни к чему не обязывает"** — снимает страх, что меня сразу заставят платить или давить на подписание договора.

5. **Простой следующий шаг** — одна большая кнопка или крупный номер телефона. Без квеста "заполните форму → дождитесь звонка → обсудите → потом запишемся".

6. **Человеческий язык** — если написано "у вас завелись клопы — это неприятно, но решаемо. Позвоните и мы разберёмся", я расслабляюсь. Если написано "проводим дезинсекционные мероприятия в жилых помещениях" — я чувствую себя на приёме у чиновника.

7. **Реальные фото специалистов** — вижу людей с именами. Мне проще открыть дверь незнакомому человеку, если я уже "познакомился" с ним на сайте.

## КРИТЕРИЙ 1: ВИЗУАЛЬНАЯ ИЕРАРХИЯ

### Как устроена у лидеров (Orkin, Molly Maid)

**Orkin.com — эталонная иерархия:**
- Hero занимает 100vh — ничто не конкурирует с главным сообщением до скролла
- Размеры шрифтов: H1 = 96px (массивный) → H2 = 48px → Body = 18-20px. Разрыв между уровнями достаточный, чтобы глаз сразу понял что важнее
- Белое пространство (whitespace) используется как инструмент: каждая секция дышит, нет ощущения "набили всё что было"
- Цвет как иерархия: красный (#CC092F) только на CTA-кнопках — больше нигде. Глаз сразу знает куда смотреть
- Информация организована по принципу "проблема → решение → доверие → действие"

**Molly Maid — иерархия через контраст:**
- Первичный цвет (#407ec9, синий) для заголовков и CTA
- Вторичный цвет (#f5c200, золото) — акцентные элементы
- Типографическая шкала: от 3rem (герой) до 0.75rem (мета-текст). Шесть чётких уровней
- "Half-border" техника: 4px верхняя граница секции в акцентном цвете — якорит раздел визуально без лишнего пространства

**Aptive Pest Control:**
- Рейтинги и телефон в самом герое — не нужно искать
- Zip-code форма в первом экране — мгновенная персонализация
- Структура: Кто мы → Что делаем → Зоны охвата → Доверие

### Ошибки российских сайтов

- **Слишком много равновесных элементов**: на profdezinfektor.ru 60+ ссылок в главном меню. Взгляд не знает куда идти
- **Карусели на герое**: ни один слайд не читается — пользователь уходит раньше, чем успевает прочитать
- **Текст на первом экране — корпоративный, не выгодный**: "Московский сервис дезинфекции / Услуги СЭС" вместо "Избавим от тараканов за 1 день — гарантия 12 месяцев"
- **Важная информация (лицензии, гарантии) — внизу**: пользователь до неё не доходит

### Формула победителей

```
Первый экран = Заголовок (боль/выгода) + Подзаголовок (уточнение) + CTA + 3-4 факта доверия
Второй экран = Услуги
Третий экран = Как работаем (шаги)
Четвёртый экран = Отзывы
Пятый экран = Ещё CTA
```

## КРИТЕРИЙ 9: ОШИБКИ ДИЗАЙНА (что отталкивает пользователей)

### Системные ошибки всех российских сайтов в нише

**1. Информационная перегрузка на главной**
- 60-100+ ссылок в меню
- 3-5 конкурирующих форм на одной странице
- Список из 200 городов в footer вместо карты или autocomplete
- Результат: пользователь не знает что делать → уходит

**2. Слайдеры-убийцы конверсии**
Каждый второй РФ-сайт начинается с карусели из 3-5 баннеров. Исследования Nielsen: конверсия каруселей ≈ 0. Пользователи их игнорируют или раздражаются.

**3. Стоковые фото насекомых крупным планом**
Таракан в 400px на главной — отталкивает. Особенно опасно для мобильных (неожиданный скролл). Лидеры (Orkin, Aptive) показывают улыбающихся технологов и чистые интерьеры.

**4. Агрессивный красный = дешевизна**
Сочетание ярко-красных кнопок + жёлтых акцентов + синего фона = кричащая, ненадёжная эстетика. Тревожит, а не вдохновляет доверие.

**5. Отсутствие мобильного мышления**
- Mega-menu на мобильном (невозможно использовать)
- Формы в 2 колонки (поля слишком маленькие)
- Отсутствие sticky телефона/кнопки
- Карточки в 3-4 колонки → горизонтальный скролл

**6. Generic корпоративный копирайт в заголовках**
"Услуги дезинфекции" — никакой эмоции, никакой выгоды.
"Официальный сайт санитарно-эпидемиологической службы" — убивает доверие (звучит как государственная контора).

**7. Сертификаты и лицензии только в JPEG внизу страницы**
Лучшие сайты показывают credential-бейджи компактно в шапке или прямо в герое.

**8. Формы без UX-обратной связи**
- "Ваше сообщение отправлено" простым текстом — пользователь не верит что что-то произошло
- Нет validation в реальном времени (сначала заполнил всё → потом узнал об ошибке)
- Нет placeholder-текстов, нет подсказок
- Кнопка "Отправить" — безличная. Должна быть "Получить расчёт стоимости"

**9. Устаревший дизайн-язык**
Flat icons из 2015, table-based layout, shadow boxes с 10px blur, border-radius 0 или 30px — всё это сигнализирует "этой компании нельзя доверять современные задачи".

**10. Нет социального доказательства выше fold**
Отзывы только внизу страницы. Рейтинги Яндекс/Google нигде не видны. Количество клиентов не упомянуто в герое.

**ФИШКА #12 — Цифры с анимацией "накрутки"**
```js
// При появлении в viewport счётчик считает от 0 до N
const counter = (el, target) => {

    if (count >= target) { el.textContent = target; clearInterval(timer); }
  }, 16);
};
```

## ИТОГОВЫЕ РЕКОМЕНДАЦИИ ДЛЯ ДЕЗ2.0

### Что ОБЯЗАТЕЛЬНО взять у лидеров

1. **Одно чёткое сообщение в герое** — не карусель, не список услуг
2. **Sticky CTA на мобильном** — кнопка звонка/заказа всегда видна
3. **Числа-факты прямо в герое**: "15 лет · 4800 обработок · Гарантия 12 мес"
4. **Форма с выбором услуги** (не просто "Ваш телефон") — персонализация
5. **Shimmer-анимация на CTA-кнопке** — чувство premium без усилий
6. **Реальные фото команды** (не стоковые технари)
7. **CSS-переменные для всей цветовой системы** — масштабируемость
8. **Fullscreen мобильное меню** с крупным шрифтом
9. **Счётчики с анимацией** при скролле
10. **WhatsApp/Telegram плавающая кнопка** — конверсия выше чем любая форма

### Одна суперфишка для Дез2.0

Адаптировать фишку Truly Nolen: **"Какая у вас проблема?" — интерактивный выбор**

```
[Тараканы] [Клопы] [Крысы] [Мыши] [Блохи] [Дезинфекция] [Плесень]
```

Пользователь кликает на свою проблему → страница показывает специфическое решение + цену.
Персонализация = доверие = конверсия.

### Технический минимум

- Tailwind CSS или CSS Custom Properties (не Bootstrap)
- Google Fonts: 1-2 шрифта максимум
- GSAP или CSS Animations для reveals
- Intersection Observer для счётчиков и fade-in
- WebP изображения + lazy loading
- Schema.org разметка (LocalBusiness, Review)
- Яндекс.Метрика + карта кликов
