**Стиль сайта:** белый фон, лаймово-зелёный акцент (#A3D044), современный, чистый, вызывающий доверие.
**Интерфейс генерации:** Nano Banana (над Gemini)
**Правило Nano Banana:** конкретные описания без пустых прилагательных ("красивый", "потрясающий"), гибрид русского описания + английских технических параметров.

**Формула промпта:** [Объект + действие] + [Место/освещение] + [Стиль] + [Технические параметры на английском]

## Общие технические теги для всех промптов в Nano Banana

Добавляй в конец любого промпта при необходимости:

- Для фото: `no watermark, no text overlay, no logo, professional photography`
- Для иллюстраций: `no gradients, flat design, vector style, no shadows`
- Для аватаров: `no accessories unless natural, no extreme expressions`
- Для всех: `suitable for commercial use, clean composition`
